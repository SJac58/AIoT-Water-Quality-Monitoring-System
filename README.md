# IoT Water Quality Monitoring System

This project implements an IoT based system to monitor water quality parameters in real time using ESP32 and cloud analytics.

## Sensors Used
- pH Sensor
- Turbidity Sensor
- DS18B20 Temperature Sensor
- Dissolved Oxygen Sensor

## Hardware
- ESP32 Microcontroller
- Sensors
- WiFi Communication

## Cloud Platform
ThingSpeak is used for real-time data storage and visualization.

## System Workflow
Sensors → ESP32 → WiFi → ThingSpeak Cloud

## Features
- Real-time water monitoring
- Cloud-based data visualization
- Remote monitoring
- Early contamination detection

## Applications
- Environmental monitoring
- Aquaculture
- Industrial water systems
- Smart cities
